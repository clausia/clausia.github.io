Getting Started

Prerequisites

  • GNU gcc compiler
  • Clang
  • GNU fortran compiler
  • TCLAP
  • IT++
  • FFTW
  • Doxygen
  • GNU Make

Ubuntu Build Guide

To install the dependencies, run the following commands.

1 sudo apt update

GNU gcc compiler

1 sudo apt install g++ clang ibc++-helpers pentium-builder

GNU fortran compiler

1 sudo apt install gfortran

IT++

1 sudo apt install libblas-dev librtlsdr-dev libncurses5-dev libitpp-dev -y

Doxygen

1 sudo apt install doxygen graphviz
